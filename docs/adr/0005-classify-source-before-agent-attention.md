# Classify source before agent attention

A fixed Git revision defines the Source Universe, with Git rather than ad hoc globbing deciding which versioned files exist; tracked files require an explicit Coverage Policy disposition even if a later `.gitignore` pattern matches them. Deterministic language classifiers assign Major, Supporting, or Excluded roles before planning, and Java DTO/VO-style Data Carriers are aggregated into meaningful Data Contracts unless validation, serialization, security, state, or business behavior promotes them, preventing file count from dominating model attention.
