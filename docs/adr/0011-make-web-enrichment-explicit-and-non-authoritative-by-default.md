# Make web enrichment explicit and non-authoritative by default

The Source Snapshot is authoritative by default, while Web and other External Knowledge Sources are disabled unless a Production Run explicitly supplies seeds, allowlists, budgets, and an Authority Level. Deterministic fetching records final location, time, digest, and truncation; external evidence cannot silently close repository-derived Major Obligations or override conflicting source facts, and network access remains constrained by scheme, host, path, redirect, IP, and size policies.
