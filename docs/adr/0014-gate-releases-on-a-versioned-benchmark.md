# Gate releases on a versioned benchmark

The Producer ships and upgrades only after passing a versioned Benchmark Corpus that covers Java, DTO-heavy code, Markdown requirements, conflicts, multi-module structure, and incremental history. Deterministic publication invariants remain absolute, while human-reviewed semantic precision, recall, concept consistency, repeated-run stability, and Mutation Cases measure probabilistic quality; changes to models, prompts, tools, classifiers, policies, profiles, frameworks, or knowledge schemas rerun the same gate.
