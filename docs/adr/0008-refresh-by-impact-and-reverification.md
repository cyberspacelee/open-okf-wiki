# Refresh by impact and reverification

Incremental Production Runs use the Knowledge Impact Graph to invalidate and reverify knowledge affected by source changes rather than rewriting the bundle from scratch. Existing Claims and Concepts cannot disappear merely because an Agent omitted them: deletion requires loss of defining evidence, no replacement evidence, renewed coverage closure, and verification, while ambiguous cases remain stale, disputed, or review-required and all successful runs still pass full publication gates.
