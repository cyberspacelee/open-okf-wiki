# Combine deterministic and semantic verification

The Deterministic Control Plane enforces mechanically decidable invariants, while independent LLM Verifiers examine evidence entailment, coverage, contradictions, concept boundaries, risk, and rendered quality through typed Verification Findings. Verifiers cannot mutate authoritative state; a configurable Acceptance Policy applies mandatory checks, risk thresholds, and human-review rules, avoiding both an inflexible all-code semantic judge and unreliable majority voting by models.
