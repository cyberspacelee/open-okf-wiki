# Fix the producer profile taxonomy

The default Producer Profile fixes a small top-level OKF taxonomy—overview, architecture, modules, flows, concepts, requirements, decisions, guides, references, and reports—while allowing the Accepted Knowledge Model to determine pages within it. Reserved indexes and logs plus coverage reports are generated deterministically, and Agents cannot invent a different top-level information architecture per run, preserving stable navigation and incremental page identity without forcing a fixed ontology of domain Concepts.
